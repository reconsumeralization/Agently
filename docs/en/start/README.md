# Get Started

Suggested reading order:

1. [Quickstart](quickstart.md): run one minimal request first.
2. [Async First](async-first.md): prefer async APIs in services, streaming paths, and workflows.
3. [Model Setup](model-setup.md): configure requesters, providers, and env variables.
4. [Agent Auto-Orchestration](auto-orchestration.md): route one Agent turn through model response or Actions with optional Skill context.
5. [Settings](settings.md): understand global, agent, and request-level settings.
6. [Project Framework](project-framework.md): separate settings, prompts, flows, and service entrypoints.
