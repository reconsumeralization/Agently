# Market Entry Analyst

Use this Skill when a task asks for a go-to-market, segment-entry, or board-level business memo.

## Requirements

- State the decision the memo supports.
- Use CRM/pipeline evidence and competitive evidence separately.
- Identify strongest opportunity, strongest risk, and immediate experiments.
- Include assumptions and missing information without blocking useful action.
- Prefer a small decision table or short bullets over long narrative.
- Do not treat example MCP data as production CRM data.
