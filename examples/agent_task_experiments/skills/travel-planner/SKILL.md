# Travel Planner

Use this Skill when a task asks for a business travel plan or itinerary.

## Requirements

- Start from the traveler's objective, timing, and constraints.
- Use host policy facts and MCP route or city facts as evidence.
- Separate fixed commitments from recommended choices.
- Include a contingency for weather, transport delay, or fatigue.
- Avoid pretending that example MCP data is live booking data.
- Keep the final plan concise enough for an operator to use.
