# Pre-4.1.3.8 Skills Orchestration Archive

These examples are archived because the 4.1.3.8 development-line Blocks
lifecycle refactor is intentionally breaking for older Skills orchestration
patterns.

The files in this directory are reference material only. They are not runnable
recommended examples for 4.1.3.8 and later and must not be adjusted to fake
compatibility with the new architecture.

Current replacements:

- Blocks lifecycle smoke: `examples/blocks/01_blocks_lifecycle_infrastructure_smoke.py`
- Direct Skills compatibility facade: `examples/skills_executor/09_runtime_planner_effort_strategy.py`
- Standard Skill registry smoke: `examples/skills_executor/01_basic_declarative_skills.py`
- Dynamic Task DAG examples: `examples/dynamic_task/`
- Current AgentExecution examples: `examples/agent_auto_orchestration/19_remote_skills_weather_event_ops.py` and `20` through `24`
